Wait, Right Here?

"Wait, Right Here?" is a brief, 2-week long class assignment where my peers and I designed a mobile application with AR functions to view local geotags. Our design features a login page, options for contributors (verified users who submit facts), and explorers (typical users). The app includes user screens such as fact submission, geotag filtering, and profile management. Given the short timeframe, we created a mid-fidelity prototype using Figma.
